# Mr. Manager - Productivity Dashboard
Built for the Mayakrit Internship Program

## Overview
A responsive task and workflow management application built with pure HTML, CSS, and JavaScript. 

## Features
* **Authentication:** LocalStorage-based login and signup simulation.
* **Kanban Board:** Drag-and-drop workflow board with sorting capabilities.
* **Team Management:** Add members and track task assignments.
* **Notes:** Real-time searchable notes database.

## Technologies Used
* HTML5
* CSS3 (Flexbox & Grid layouts)
* Vanilla JavaScript (DOM manipulation, LocalStorage)

## DSA Concepts Implemented
* **Arrays & Objects:** Used to structure and store LocalStorage data.
* **Searching:** Linear search applied for real-time notes filtering.
* **Sorting:** `.sort()` algorithm applied to order tasks by Priority weights or Due Date.
